# Njaanuary Survival Guide
A lightweight Python tool built to track spending during the toughest month of the year.

### Features
* **Budget Guard:** Prevents logging expenses that exceed your income limit
* **Persistent Storage:** Uses JSON to keep track of every shilling spent.

## How to Run
1. Ensure you have Python installed.
2. Clone this folder and open your terminal inside it.
3. Run the application:
   - **Windows:** `python database.py` or `py database.py`
   - **Mac/Linux:** `python3 database.py`
